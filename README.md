This repository contains the MML Game Engine - a superset of the core MML elements and libraries available at [https://github.com/mml-io/mml](https://github.com/mml-io/mml). 

This repository is a fork of that repository and is intended to be kept up-to-date from that upstream.

## Quick Start

1. Clone the repo
1. `npm install`
1. `npm run iterate` Builds and starts incrementally building package artefacts from sources
  * Servers should start for examples:
    * [http://localhost:3030](http://localhost:3030) - MML Game Examples
