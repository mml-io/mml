# Overlay Health Bar

Demonstrates animated health bar UI using `m-overlay` with `layout="yoga"`.

- Animated health bar that smoothly transitions when taking damage
- Random damage button (5-50 HP)
- Heal button to restore health
- Color changes based on health level (green → yellow → red)
- Flexbox layout using Meta's Yoga layout engine

Click "Take Damage" to see the health bar animate down with the number updating.
