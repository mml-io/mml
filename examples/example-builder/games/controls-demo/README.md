Controls Demo

Shows keyboard, mouse, and gamepad inputs in real time. Also visualizes left/right sticks.

How to use:
- Interact with keyboard (WASD, arrows, Space, Shift), mouse clicks, and any connected gamepad.
 

Files:
- src/index.mml: Scene and overlay UI with input bindings.
- src/index.ts: Optional logging of input events.
- src/scripts.json: Empty; no extra systems required.
