# Top Down Game

This is a simple top down game example.

## Running the game

Make sure you have the mml-cli installed globally. To use the local build of the mml-cli, see the [mml-cli README](../../packages/mml-cli/README.md). You can check if it's installed globally by running `mml` which should show the help screen.

```bash
npm run dev
```

This runs the game on `http://localhost:3004`. An MCP server is also running on `http://localhost:3004/mcp/sse`.
