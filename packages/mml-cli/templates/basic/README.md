# MML starter

Commands:
- `mml editor` to watch and serve at http://localhost:3003
- `mml serve` to watch/build and serve at http://localhost:3004
- `mml build` for a one-shot build, or `mml build --bundle` for a single-file dist in `dist/`

Project layout:
- `src/main.mml` ‚Äî starter scene with physics + character controller
- `src/main.ts` ‚Äî helper script for spawning crates and controller events
- `src/scripts.json` ‚Äî loads math + physics systems
- `assets/` ‚Äî shared models/textures served at `/assets`
